## My Building Philosophy

Three things I've decided to actually believe, not just say:

**1. Build before you're ready.**
Waiting until you fully understand something before building with it is how you never build anything. I write projects that are slightly beyond what I know. That gap is where learning happens.

**2. Consistency over intensity.**
A 4-hour coding binge followed by 5 days off is worse than 45 minutes every single day. This repo gets a commit as often as humanly possible.

**3. Clarity over comparison.**
Someone on Twitter just shipped an ML model at age 19. Someone else has been coding for a decade. Neither of those facts changes what I need to do today. This journey is measured against my Day 1 self вАФ nothing else.
